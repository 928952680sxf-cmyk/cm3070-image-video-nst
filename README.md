# cm3070-image-video-nst
CM3070 project: hybrid image/video neural style transfer with style-collapse diagnosis, strict same-frame audit, and reproducible evaluation artifacts.

Core deliverables only.

## Included

- `reports/cm3070_final/main.tex`: report source.
- `reports/cm3070_final/references.bib`: bibliography database.
- `reports/cm3070_final/assets/`: report figure assets (split panels + metric plots).
- `reports/review/`: template/report visual parity compares and sampled rendered pages.
- `experiments/`: core CSV metrics and protocol audit files.
- `video/`: strict same-frame side-by-side mp4 outputs.
- `scripts/`: reproducible training/evaluation/report build scripts.
- `configs/`: key experiment configs.
- `src/nst/`: core NST implementation modules.
